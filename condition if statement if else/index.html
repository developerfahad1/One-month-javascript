✅ What is an if Statement?

if statement ka use hota hai kisi condition ko check karne ke liye. Agar condition true ho to code chalega, warna skip ho jayega

✅ if...else

Agar condition true hai to if wala code chalega, warna else wala.

✅ if...else if...else

Multiple conditions check karne ke liye use hota hai.
